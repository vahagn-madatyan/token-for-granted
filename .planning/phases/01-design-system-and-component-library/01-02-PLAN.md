   ```ts
     animate: {
       boxShadow: [
         '0 0 20px rgba(143, 245, 255, 0.1)',
         '0 0 40px rgba(143, 245, 255, 0.3)',
         '0 0 20px rgba(143, 245, 255, 0.1)',
       ],
       transition: { duration: 2, repeat: Infinity },
     },
   }
   ```

2. Create `src/components/animations/useCountUp.ts`:
   - Hook signature: `useCountUp(target: number, duration: number = 2000): number`
   - Uses `useState` for current value and `useEffect` with `requestAnimationFrame`
   - Applies easeOutExpo easing: `function easeOutExpo(t: number): number { return t === 1 ? 1 : 1 - Math.pow(2, -10 * t) }`
   - On mount, starts animation from 0 to `target` over `duration` ms
   - Returns the current animated value (as a number, caller formats it)
   - Cleans up animation frame on unmount
   - If `target` changes, restarts animation

3. Create `src/components/animations/useTypewriter.ts`:
   - Hook signature: `useTypewriter(text: string, speed: number = 50): { displayText: string; isComplete: boolean }`
   - Uses `useState` for current index and `useEffect` with `setInterval`
   - Each interval tick increments character index by 1
   - Returns `text.slice(0, currentIndex)` as displayText
   - Sets `isComplete` to true when `currentIndex >= text.length`
   - Clears interval on unmount or when complete
   - If `text` changes, resets to beginning

4. Create `src/components/animations/useGlitch.ts`:
   - Hook signature: `useGlitch(): { ref: React.RefObject<HTMLElement>; triggerGlitch: () => void; isGlitching: boolean }`
   - Uses `useState` for isGlitching and `useRef` for the element
   - `triggerGlitch()` adds the `.glitch` CSS class to the ref element, sets isGlitching true
   - After 300ms timeout, removes the class and sets isGlitching false
   - Cleans up timeout on unmount
   - Alternative simpler approach: just toggle a boolean state that the consumer can use to conditionally apply `className="glitch"` -- either approach is fine
